# 文本转二维码生成器

一个简洁易用的 Web 应用，可以将任意文本转换为二维码图片。

## 功能特点

- 📝 支持任意文本转二维码
- 🖼️ 高质量二维码生成（256x256）
- 💾 一键下载二维码图片
- ⌨️ 支持 Ctrl+Enter 快捷键
- 📱 响应式设计，支持移动端
- 🎨 美观的渐变色界面

## 使用方法

1. 在浏览器中打开 `index.html`
2. 在文本框中输入要转换的内容
3. 点击"生成二维码"按钮或按 Ctrl+Enter
4. 点击"下载二维码"保存图片

## 技术栈

- HTML5
- CSS3
- JavaScript
- QRCode.js

## 本地运行

直接在浏览器中打开 `index.html` 文件即可，无需安装任何依赖。
